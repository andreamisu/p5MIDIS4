# p5MIDIS4
personal p5.js wrapper for Native Instruments S4 MKII interface

## Usage
Dropins actual sourcecode inside function Setup() in an already predefined p5.js project.
It requires Native Instruments Controller Editor to be available and running on the machine.

## Implmentation
- each 4 Decks are mapped in different scenarios.
- values mapped:
	- low, mid, hi EQ
	- filter
	- fader
	- audio cue
	- play cue
